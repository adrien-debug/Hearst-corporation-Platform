# HearstCorporation Platform - Documentation Technique & UX

Ce dossier contient la structure complète, la documentation UX/UI d'intégration, et les schémas techniques JSON de la plateforme HearstCorporation.

Chaque page fonctionnelle de la plateforme (admin & utilisateur) est décrite dans un document distinct, avec wireframe, UX détaillée et son schéma JSON strict pour recodage, migration ou audit.

Pour chaque module vous trouverez des fichiers :
- `.md` : Plan UX très détaillé, flow, comportements, microcopy, states et règles.
- `.json` : Structure de donnée/JSON schema attachée à la page.
